# 日常配置文件

这些是我在Linux下工作时候使用时一些软件的配置文件，希望各位能在配置这些软件的时候进行参考。

## zsh

在日常使用中使用zsh作为默认的shell环境，使用[oh-my-zsh](http://ohmyz.sh/)作为日常的的zsh的配置

在这个配置中，需要这些插件：

1. zsh-autosuggestions
2. zsh-syntax-highlighting
3. thefuck
4. fzf
5. docker
6. Anaconda
